# fast_mapping_4DSTEM

This repository contains Jupyter notebooks and datasets for fast mapping and analysis of 4D-STEM (Four-Dimensional Scanning Transmission Electron Microscopy) data for the manuscript "Fast 4D-STEM-based phase mapping for amorphous and mixed materials" by Andreas Werbrouck, Nikhila C. Panaramana, Xiaoqing He and Matthias J. Young. The manuscript is currently under revision. 

## Structure

- **Notebooks (`*.ipynb`)**: Workflows for processing and visualizing different material datasets, including:
  - Battery materials (NMC/LGPS)
  - Oxide materials (TiO2)
  - 2D materials (WS2)
  - Metallic glasses / amorphous materials (ZrCuAl)
- **`data/`**: Subdirectory for storing datasets (e.g., `.h5`, `.dm4` files). Note that it is empty in this repository. The ZrCuAl dataset should be obtained from Dr. Koji Kimoto, the WS2 dataset can be obtained here: [https://github.com/py4dstem/py4DSTEM_tutorials/blob/main/notebooks/orient_strain_01_WS2.ipynb], the TiO2 and NMC datasets can be obtained here: [https://doi.org/10.5281/zenodo.20366781]
- **`factorizations/`**: Subdirectory for storing saved NMF/PCA factorization models and results.
- **`figures/`**: Subdirectory for output graphs and visualizations generated by the notebooks.
